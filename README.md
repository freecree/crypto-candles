# Introduction

Component that visualize candlestick and random generated buy / sell signal data for USDT/BTC pair on a chart.

# Installation

`npm install crypto-candle`

# Usage

```js
import { CryptoChart } from 'crypto-candles';

function App() {
  return (
    <>
      <CryptoChart />
    </>
  );
}

export default App;
```
